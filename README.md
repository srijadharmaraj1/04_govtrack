# 🏛️ GovTrack Tamil Nadu — Complete Setup Guide

## What This Does
An AI agent that searches for Tamil Nadu government officials,
saves them locally, and shows them on a live dashboard.

## 🔑 STEP 1 — Add Your Serper API Key

1. Go to https://serper.dev → sign up free
2. Copy your API key
3. Open `.env` and replace:
   ```
   SERPER_API_KEY=your_actual_key_here
   ```

---


## 🐍 STEP 2 — Python Environment & Dependencies

Open PowerShell inside the `04_govtrack` folder:

If using conda (recommended):
```powershell
conda activate eag_v3_env
```

Then install dependencies:
```powershell
pip install mcp[cli] httpx python-dotenv
```

---


## ⚙️ STEP 3 — Test MCP Server

```powershell
mcp dev govtrack_server.py
```

This opens MCP Inspector at [http://localhost:5173](http://localhost:5173)
You can test all 3 tools manually there!

---


## 🌐 STEP 4 — Set Up Dashboard (Frontend & Bridge)

Open a NEW PowerShell window:
```powershell
cd 04_govtrack/dashboard

# Install dependencies
npm install

# Start bridge server (receives MCP pushes)
node bridge.cjs
```

Open ANOTHER PowerShell window:
```powershell
cd 04_govtrack/dashboard

# Start React dashboard
npm run dev
```

Open your browser at [http://localhost:3000](http://localhost:3000)
You should see the GovTrack dashboard!

## 🤖 STEP 5 — Connect to Claude Desktop

1. Open Claude Desktop
2. Go to Settings → Developer → Edit Config
3. Add this to claude_desktop_config.json:

```json
{
  "mcpServers": {
    "govtrack": {
      "command": "python",
      "args": ["C:\\Users\\YourName\\Desktop\\govtrack\\govtrack_server.py"],
      "env": {
        "SERPER_API_KEY": "your_key_here"
      }
    }
  }
}
```

4. Restart Claude Desktop
5. You should see a 🔌 icon — GovTrack tools are connected!

## 🎯 STEP 6 — Demo Prompt

Copy-paste this into Claude Desktop:

```
You have access to GovTrack tools. Please do the following:

1. Use fetch_authority to search for:
   - "Collector of Coimbatore"
   - "SP of Coimbatore"  
   - "MLA of Coimbatore North"

2. Use crud_profile to save each result to local storage

3. Use crud_profile to read all saved profiles

4. Use push_to_dashboard to send all profiles to the dashboard
   with the message "Coimbatore District Officers — Live"

After this, I should see all 3 profile cards at localhost:3000
```

## ✅ What Happens

```
Claude reads your prompt
  → calls fetch_authority("Collector of Coimbatore")
    → searches Serper API → returns profile data
  → calls crud_profile("create", profile)
    → saves to data/profiles.json
  → repeats for SP and MLA
  → calls push_to_dashboard([...all profiles])
    → POSTs to bridge server at :3001
      → React dashboard polls :3001 every 2 seconds
        → Cards appear LIVE on your dashboard! 🎉
```

## 🔧 Troubleshooting

| Problem                   | Fix                                         |
|--------------------------|----------------------------------------------|
| `mcp: command not found` | `pip install mcp[cli]`                       |
| Dashboard blank          | Make sure bridge.cjs is running on :3001     |
| No results from search   | Check SERPER_API_KEY in .env                 |
| Claude can't see tools   | Restart Claude Desktop after config change   |
